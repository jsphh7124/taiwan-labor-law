# Taiwan Labor Law Teaching Skill

台灣勞動法規教學與案例分析工具,專為組織行為與人力資源管理教學設計。

## 功能概述

本 plugin 提供三個核心功能,協助教師進行勞動法規教學:

1. **案例分析** - 深入分析資遣、解僱、工時等勞動法案例
2. **合規檢查** - 檢視人資政策、勞動契約的法規合規性
3. **情境生成** - 產生適合課堂討論的教學案例

## 斜線命令

### `/analyze-dismissal-case` - 案例分析

分析勞動法相關案例,提供法律事實認定、適用法條分析、判決預測與教學討論點。

**使用方式**:
```
/analyze-dismissal-case

[貼上案例描述或提供檔案路徑]
```

**輸出內容**:
- 案例摘要與法律爭點
- 適用法條分析(勞基法、勞資爭議處理法等)
- 判決預測與理由
- 教學討論問題
- 相關判例參考

---

### `/check-labor-compliance` - 合規檢查

檢查人資政策、勞動契約、工作規則等文件的法規合規性。

**使用方式**:
```
/check-labor-compliance

[貼上政策內容或提供檔案路徑]
```

**輸出內容**:
- 🟢 合規項目
- 🟡 灰色地帶(有爭議風險)
- 🔴 違法風險項目
- 具體改善建議與法條依據

---

### `/generate-teaching-scenario` - 教學情境生成

根據指定主題生成適合課堂討論的勞動法教學案例。

**使用方式**:
```
/generate-teaching-scenario 資遣
/generate-teaching-scenario 職業災害
/generate-teaching-scenario 工時爭議
```

**常見主題**:
- 資遣與解僱
- 工時與加班
- 職業災害
- 勞動契約
- 假勤管理
- 工資爭議

**輸出內容**:
- 完整情境描述(背景、角色、事件)
- 分層次的討論問題(事實認定→法律分析→判斷預測→批判反思)
- 教師參考解答
- 延伸討論方向(組織行為、人力資源管理觀點)

## 自動觸發 Skill

當您詢問台灣勞動法規相關問題時,`taiwan-labor-law-expertise` skill 會自動載入,提供:

- 勞動基準法重點條文解析
- 資遣與解僱法定要件
- 工時與加班規定
- 職業災害補償
- 勞資爭議處理程序

**觸發關鍵詞**:
台灣勞動法、勞基法、資遣要件、解僱程序、工時規定、職業災害、勞資爭議、勞動契約、加班費、特別休假、工資爭議

## 設計特色

✅ **台灣法律體系導向** - 基於大陸法系的法律解釋方法
✅ **教學情境優先** - 輸出格式適合課堂討論與案例教學
✅ **證據導向** - 引用具體法條、判例與學術文獻
✅ **批判思考** - 鼓勵多角度分析,避免單一標準答案
✅ **跨領域整合** - 連結組織行為與人力資源管理理論

## 適用對象

- **教師** - 組織行為、人力資源管理、勞動法規課程
- **HR 從業人員** - 人資政策制定、勞資關係管理
- **學生** - 商管、法律相關科系學生

## 使用範例

### 範例 1: 分析資遣案例

```
/analyze-dismissal-case

某公司員工小明因連續三個月績效考核未達標準,公司擬予以資遣。
小明已在公司任職 2 年 3 個月,公司提前 20 日通知並願意給付資遣費。
請問公司此舉是否合法?
```

### 範例 2: 檢查加班政策

```
/check-labor-compliance

公司規定員工每日工作時間為 09:00-18:00,中午休息 1 小時。
若業務需要加班,需事前申請,加班費以平日每小時工資 1.34 倍計算。
```

### 範例 3: 生成教學情境

```
/generate-teaching-scenario 職業災害補償
```
